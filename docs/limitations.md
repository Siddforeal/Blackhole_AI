# Limitations

BugIntel AI Workbench is an early research prototype.

## Current Limitations

- Web/API only.
- No live browser execution yet. Browser planning, Playwright execution preview, capture-result evidence import, and report rendering are supported.
- No Android or iOS analysis yet.
- No LLM integration yet.
- No Burp/HAR importer yet.
- Response-diff results require manual validation.
- The tool does not prove vulnerabilities automatically.

## Future Work

- Playwright browser traffic capture
- HAR/Burp import
- AI planning layer
- Report generator
- Android APK static analysis
- iOS IPA/plist analysis
- Dashboard UI
